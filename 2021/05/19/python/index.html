计算机发展遵循摩尔定律（指数方式）


源代码：采用某种编程语言写的计算机程序，人类可读

目标代码：计算机执行的代码


编译：将源代码一次性转换为目标代码，后运行（编译器）

静态语言：C，C++，java

程序运行速度快，运行时不需要源代码


解释：将源代码逐条转换为目标代码同时逐条运行（解释器）interpreter

脚本语言：python，javascript，php

维护灵活，运行时需要源代码

程序基本编写方法：IPO

input process output


缩进表示格式框架:需长度一致


注释：

单行：#

多行：以’’’开头和结尾


保留字in：判断是否与列表中的某个元素相同


评估函数：eval

去掉字符串的双引号或单引号，变为别的数据类型


计算机技术的演进过程：

2017 人工智能时代 人类的问题
2008-2016 复杂信息系统时代 数据问题
1981-2008 网格和视窗时代 交互问题
1946-1981 计算机系统结构时代 计算机能力问题

涉及到计算机内存、接口等工作还是需要c语言

编程语言 学习内容 语言本质 解决问题
C 指针，内存，数据类型 理解计算机系统结构 性能
JAVA 对象，跨平台，运行时 理解主客体关系 跨平台
C++ 对象，多态，继承 理解主客体关系 大规模程序
VB 对象，按钮，文本框 理解交互逻辑 桌面应用
PYTHON 编程逻辑，第三方库 理解问题求解 各类问题

python计算生态=标准库（随解释器直接安装）+第三方库（需安装）

模块：库library 包package 模块module


编程语言种类：机器语言 汇编语言 高级语言 超级语言


指定窗体大小位置：

turtle.setup(width,height,startx,starty) （后俩可省，默认在正中心）

（setup不是必需的）


turtle.goto 海龟去哪

（绝对坐标中中间为0，0）


RGB:

可用0-255整数或0-1的小数

white(255,255,255)(1,1,1)

black(0,0,0)(0,0,0)

blue(0,0,255)


改变颜色模式：

turtle.colormode(mode) mode：1.0 255

默认采用小数值，可切换为整数值


库引用：

import 库名

库名.函数名（参数）


from 库名 import 函数名或* （此方法可能重名）


import 库名 as 别名 （给外部库起别名）


tuerle画笔控制函数：

turtle.penup（） pu：抬起画笔

turtle.pemdown（） pd：降下画笔

turtle.pensize（） width：画笔宽度

turtle.pencolor(“purple”) 颜色字符串

turtle.pencolor(0.63,0.13,0.94) RGB小数值

turtle.pencolor((0.63,0.13,0.94)) RGB原组值


turtle运动控制函数：

turtle.forword(d) fd:向前走距离d，单位为像素,指海龟坐标

turtle.circle(r,extent=None)：绘制半径为r，extent角度的弧形（extent省略时默认360）

圆心默认在海龟左侧r位置


turtle方向控制函数：

turtle.setheading(angel) seth：绝对角度转向

turtle.left(angel)：海龟角度左转

turtle.right(angel)：海龟角度右转


循环语句：

for 变量 in range(次数):


print(“hello”,i)

用逗号将输出的各种变量分割，输出时中间会加空格


range（N） 产生0到N-1的整数序列

range（m，n） 产生m到n-1的整数序列


turtle.done() 需要手动关闭窗体


python中表示数的数据类型有三种

整数 浮点数 复数

从左到右依次变宽，计算时会往宽的数据类型转换


python中整数无限制

二进制：0b 0B开头

八进制：0o 0O开头

十进制：0x 0X开头


浮点数的取值范围和小数精度存在限制，但可以忽略，因为足够大


用二进制表示小数，可以无限接近，但不能完全相同，所以计算小数不精确，但通常情况下可以忽略

4.3e-3为0.0043 9.6E5为960000.0


a+bj 复数类型

z.real取实部

z.imag取虚部


10/3==3.3333333

10//3==3

10%3==1

x**y 幂运算 x的y次方

还有复合运算符 += -= *= 等


abs(-10.01)==10.01

divmod(10,3)==(3,1)

pow(x,y[,z]) []中表示可省略 x的y次幂后除z的余数 pow（3，pow(3,99),10000） 用来求后几位

round(x,d) 对x四舍五入，d为小数截取位数 用于浮点数的比较运算：round(0.1+0.2,1)==3

max(1,9,5,4,6)==9

min(1,9,5,4,6)==1

int(123.45)==123 int(“123”)==123

float(12)==12.0 float(“1.23”)==1.23

complex(3)==3+0j


一对单引号或双引号括住表示单行字符串

一对三单引号或三双引号括住表述多行字符串 既是注释又是字符串


字符串切片

[M:N] M缺失表示到开头，缺失表示到结尾 到N之前，没有N

[M:N:K] K为步长


转义字符：如\”

特殊：\b \n(换行) \r（光标回到本行首）


字符串操作：

x+y 连接

nx xn 把字符串x复制n次

x in s x是s的子串返回True

len（“一二三456”）==6 汉字算一个字符

str（x） 把其他任何类型转化为字符串

hex(x) oct(x) 整数x的十六进制或八进制小写形式

chr(u) ord(x) 字符x和unicode编码u之间转换


Unicode编码：（python使用）

统一字符编码，几乎覆盖所有字符的编码方式

从0到1114111（0x10FFFF）


str.lower() strupper() 全部字符小写或者大写 “Abcd”.lower()==”abcd”

str.split(sep=None) “A,B,C”.split(“,”)==[‘A’,’B’,’C’]

str.count(sub) “an apple a day”.count(‘a’)==4

str.replace(old,new) 替换

str.center(width[,fillchar]) 居中（宽度，添加的字符）


槽{}

“{1}:计算机{0}的cpu占用{2}”.format(“1”,“11：11”,”99”)


“{0:=^20}”.format(“dasda”) :引导符 =用于填充的字符 <左对齐^居中对齐 20宽度

“{0:,.2f}”.format(123.456) ，数字的千位分隔符 .2小数或字符串最大长度 f浮点数类型


import time python中处理时间的标准库

time.time() 获取当前时间戳

time.ctime() 返回字符串的当前时间

time.gmtime() 获取当前时间 表示为计算机可处理的格式

time.strftime(tpl.ts) 与上一条一起用：%Y %m %d 等

time.strptime(str.tpl) 与上一条相反 用于输入时间

time.perf_counter() 返回一个cpu时间 可两个相减后求时间

time.sleep（s） 让程序等待s秒
